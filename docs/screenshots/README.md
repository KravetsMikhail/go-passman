# Screenshots of the web interface

Place screenshot files here so they appear in the main README:

| File | Description |
|------|-------------|
| `web-list.png` | Main list of entries (home page) |
| `web-unlock.png` | Unlock form (encrypted vault) |
| `web-add.png` | Add new entry form (optional) |

To capture real screenshots: run `go-passman -w`, open http://127.0.0.1:8080 in a browser, and save the images with the names above.
