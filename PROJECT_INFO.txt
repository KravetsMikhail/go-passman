================================================================================
                              OVERVIEW
================================================================================

go-passman is a secure, lightweight command-line password manager written in Go.
It provides a simple interface for storing, managing, and encrypting passwords
locally on your machine.

LANGUAGE:            Go 1.19+
SINGLE BINARY:       Yes (no runtime dependencies)

================================================================================
                              CORE FEATURES
================================================================================

✅ Password Management
   - Add new passwords (manual or auto-generated)
   - Remove passwords with confirmation
   - Update existing passwords
   - List all stored services
   - Copy passwords to clipboard

✅ Security
   - AES-256-GCM encryption (military-grade)
   - PBKDF2-SHA256 key derivation (100,000 iterations)
   - Secure password input (hidden on Unix-like systems)
   - Master password protection

✅ Utility Features
   - Password generation with customizable options
   - Open vault in text editor for manual editing
   - Show vault status and location
   - Encrypt/decrypt vault on demand

✅ Portability
   - Single executable, no external dependencies
   - Cross-platform: Windows, Linux, macOS
   - Vault stored in executable directory
   - USB stick friendly

================================================================================
                              COMMANDS
================================================================================

go-passman add                    Add new password (manual or generated)
go-passman remove <service>       Remove a password
go-passman copy <service>         Copy password to clipboard
go-passman list                   List all services
go-passman update                 Update existing password
go-passman open [editor]          Open vault in text editor
go-passman encrypt                Encrypt vault with master password
go-passman decrypt                Remove vault encryption
go-passman status                 Display vault information
go-passman path                   Show vault file location
go-passman --help                 Display help
go-passman --version              Display version

================================================================================
                              SECURITY
================================================================================

Encryption Algorithm:          AES-256-GCM
  - 256-bit key size
  - Galois/Counter Mode (authenticated encryption)
  
Key Derivation:                PBKDF2-SHA256
  - 100,000 iterations
  - 16-byte random salt
  - 32-byte key length
  
Nonce:                         12 random bytes per encryption
Random Number Generation:       Cryptographically secure (crypto/rand)

File Permissions:              0600 (owner read/write only)
Master Password:               Never stored or logged
Encrypted Vaults:              Base64-encoded binary data

Security Considerations:
  ✅ NIST-approved algorithms
  ✅ Proper key derivation
  ✅ Secure random generation
  ✅ No password logging
  ✅ File permissions restricted
  ⚠️  Windows password input not hidden (OS limitation)
